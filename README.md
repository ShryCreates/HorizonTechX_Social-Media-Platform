# 🚀 SocialHub

A modern full-stack social media platform where users can connect, share posts, upload images, follow other users, and interact through likes and comments.

## 📌 Features

### 🔐 Authentication

* User Registration
* User Login & Logout
* JWT-based Authentication
* Protected Routes

### 👤 User Profile

* View User Profiles
* Profile Bio
* Followers & Following Count
* Follow / Unfollow Users

### 📝 Posts

* Create Text Posts
* Upload Image Posts
* Edit Posts
* Delete Posts
* View Feed

### ❤️ Social Interactions

* Like Posts
* Comment on Posts
* Delete Comments
* User Search Functionality

### 🎨 User Experience

* Responsive Design
* Dark Mode Support
* Modern UI with Glassmorphism Effects
* Interactive Animations
* Toast Notifications

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/SOCIAL_MEDIA_PLATFORM.git
cd SOCIAL_MEDIA_PLATFORM
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Create Environment Variables

Create a `.env` file in the root directory:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

### 4. Start the Server

```bash
node server.js
```
